# Changelog

All notable changes to this project will be documented in this file.

## [1.0.2] - 2026-01-21
### Added
- Initial release of `wave_animation` package.
- `WaveWidget` for customizable wave animations.
- `WavePainter` for multiple wave layers with gradient colors.
- `WaveMotionPattern` enum with `classic`, `flowField`, `ribbonDrift`, `flowFieldLoop`, `ribbonLoop`.
- Null safety for all properties.
- AnimatedBuilder with speed and visibility control.
- Test file for widget instantiation and painter class.
